# 1: What is the difference between var, let, and const?

## Answer: Difference is var maintain function scope while let and const are block scoped, and var can be reassign and redeclare also can access before initialization but return (undefined), let can't be redeclare, but can reassign, const can't redeclare or reassign if they have primitive data, let and const maintain TDZ (Temporal Dead Zone) that's why can't access before initialization.
